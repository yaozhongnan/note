# 阿里云项目部署

## 问答项目

前端项目打包后的文件可以直接仍到服务器的 ~/ROOT 文件夹下，它会自动映射到 tomcat 容器内部

```bash
# tomcat
docker run -d -p 8080:8080 -v ~/ROOT:/usr/local/tomcat/webapps/ROOT 镜像ID
```

后端项目在 ~/wd-server 目录下，通过 pm2 start index.js 即可。

数据库通过 docker 启动，数据库名称为 wd_app

```bash
# mongo 数据库启动命令
docker run -itd --name mongo -p 27017:27017 镜像ID
```
